=========================
Trossen SDK Documentation
=========================

Documentation for the Trossen SDK.
The Trossen SDK is a C++ SDK for recording robot demonstrations with Trossen AI Kit arms, Stereolabs ZED cameras, RealSense cameras, and the SLATE mobile base.
Episodes are recorded to the **TrossenMCAP** format, visualized in **Foxglove Studio**, and converted to **LeRobot V2** format for training.
