=========================
Trossen SDK Documentation
=========================

The Trossen SDK is an open, extensible C++ framework for recording robot demonstrations.
One pipeline captures synchronized data across arms, cameras, and mobile bases, with first-class support for Trossen hardware and clean extension points for bringing your own.
Pluggable hardware, producer, and storage registries let you swap devices, streams, and output formats without forking core code.
Record once at full fidelity, visualize, replay, and export to the training format of your choice.
